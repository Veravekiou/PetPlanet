# Pet Planet

Pet Planet is a static pet shop website UI built with HTML, CSS, and a small amount of vanilla JavaScript.

## Features

- Home page with hero section and services
- Product category pages for dogs, cats, birds, and fish
- Adoption listing page with pet cards
- Contact page UI
- Login and sign-up interface mockup
- Responsive CSS for desktop and smaller screens

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome icons

## Project Type

This is a frontend/static website project. It does not include a backend, database, real authentication, payment system, or working contact form submission.

## Project Structure

- `index.html` - home page
- `pages/` - all secondary HTML pages
- `css/` - shared and page-specific stylesheets
- `images/` - image assets

## How to View

Open `index.html` in a browser, or deploy the repository with GitHub Pages.
